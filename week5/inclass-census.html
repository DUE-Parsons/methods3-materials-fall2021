In-class Exercises

Join census data

  1. In QGIS, open the CSV of census data (nys-owner-renter/nys-owner-renter.csv). Use Layer > Add Layer > Add Delimited Text Layer with No geometry selected.
  2. Open the shapefile in nys_census_tracts QGIS.
  3. Join the spreadsheet to the shapefile. You should be able to use GEO_ID and AFFGEOID.
  4. Make a choropleth showing the percentage of housing units that are vacant.
  5. Try making another choropleth.
  6. What if we only wanted to see this map for NYC? Save the joined layer as a new shapefile, then clip that layer to the Borough Boundaries shapefile. Alternatively, you could use select by location to find all tracts that overlap with the borough boundaries. Style this layer with a graduated style.
