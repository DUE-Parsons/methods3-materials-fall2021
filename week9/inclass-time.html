Temporal Controller

  1. Download this data.
  2. Open the CSV file in the earthquakes folder in QGIS.
  3. Open the Temporal Controller panel in QGIS (View > Panels > Temporal Controller).
  4. Open the earthquake layer properties (right-click, Properties).
  5. Go to the Temporal tab.
  6. Check the box next to Temporal to enable it, then select Single Field with Date/Time from the Configuration dropdown. Set the Field to time.
  7. Click Ok to close the properties window.
  8. Change some of the settings in the Temporal Controller panel to make an animation using the earthquakes data. You may need to change the frame rate (in the settings) and the step length to get a smooth animation.
