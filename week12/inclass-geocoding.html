Geocoding Example

  1. For this exercise, we'll geocode the stores that got new tobacco licenses in Brooklyn in 2021 - the dataset that we processed in R. I have purposefully selected a small number of addresses for this exercise. Geocoding is a laborious process, and it takes a while no matter which service you use.
  2. Open QGIS and install the MMQGIS Plugin
  3. In the new MMQGIS dropdown, select Geocode CSV with Web Service
  4. Open the BK_tobacco_licenses_2021.csv
  5. Input CSV = BK_tobacco_licenses_2021.csv file that you wrote out from R.
  6. Select Web Service = "US Census Bureau"
  7. Name your Output File Name and Not Found Output List, and save them in your class12/data/processed folder.
  8. Click Apply
  9. Add a baselayer using QuickMap Web Service to see where the geocoded points are
  10. Open the Not Found Output List to see if any stores were not successfully geocoded.
  11. If there were, you can look at the address to see if you can determine the error in the address of the ungeocoded addresses. You may want to correct the Not Found Output addresses and geocode again.
