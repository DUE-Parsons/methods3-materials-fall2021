Spatial Joins

  1. We will use a spatial join to count the number of schools, and the number of elementary schools in each school district in New York.
  2. Open the School Districts shapefile in QGIS.
  3. Open the Public School Points shapefile.
  4. The schools shapefile has an indicator, elem_num that is 1 if it is an elementary school, and zero otherwise. We'll use this indicator to add up the number of elementary schools.
  5. Select Join Attributes by Location (Summary) from the search bar.
  6. Input layer = school districts
  7. Join layer = Public schools points
  8. Fields to summarise, select elem_num
  9. Summaries to calculate, select count and sum to calculate the number of schools and the number of elementary schools
  10. Make a choropleth with the result.
